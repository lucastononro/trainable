# delegate-task

Delegate a task to a specialized sub-agent. The sub-agent runs to
completion and returns its output text. Use this to break complex
workflows into steps handled by specialist agents.

## When to use
Hand off a focused subtask to a specialist sub-agent and wait for its result.
